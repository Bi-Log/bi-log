Logarithms


Definition of Logarithms

Logarithms are mathematical functions that describe the relationship between exponents and bases. Understanding their basic properties is fundamental in various fields of mathematics. Here are some key properties:


Basic Properties of Logarithms

If b > 0 and b \(\neq\) 1, then:

1. \(\log_{b}{1}=0\), because \(b^0=1\)

2. \(\log_{b}{b}=1\), because \(b^1=b\)

3. \(\log_{b}{b^x}=x\), and \(b^{\log_{b}{x}}=x\) (Inverse Property)


Examples of Property 1


Example 1

\(\log_{2}{1}\) using the property \(\log_{b}{1}=0\) (property 1). If you raise 2 to the power of 0, the result is 1. Therefore,

\[\log_{b}{1}=0\]

\[b^0=1\]

\[\log_{2}{1}=0\]

\[2^0=1\]

Example 2

\[\log_{b}{1}=0\]

\[b^0=1\]

\[\log_{16}{1}=0\]

\[16^0=1\]

Example 3

\[\log_{b}{1}=0\]

\[b^0=1\]

\[\log_{25}{1}=0\]

\[25^0=1\]

Examples of Property 2


Example 1

\(\log_{6}{6}\) using the property \(\log_{b}{b}=1\) (property 2), we can simplify \(\log_{6}{6}\) to 1

\[\log_{b}{b}=1\]

\[b^1=b\]

\[\log_{6}{6}=1\]

\[6^1=6\]

Example 2

\[\log_{b}{b}=1\]

\[b^1=b\]

\[\log_{100}{100}=1\]

\[100^1=100\]

Example 3

\[\log_{b}{b}=1\]

\[b^1=b\]

\[\log_{2}{2}=1\]

\[2^1=2\]

Examples of Property 3


Example 1

\(\log_{2}{2^4}\) using \(\log_{b}{b^x}=x\) (property 3), we can simplify \(\log_{2}{2^4}\): according to the property, the result is the exponent, which is 4.

\[\log_{b}{b^x}=x\]

\[log_{2}{2^4}=4\]

Example 2

\[\log_{b}{b^x}=x\]

\[\log_{5}{125}=\log_{5}{5^3}=3\]

Example 3

\[\log_{b}{b^x}=x\]

\[\log_{7}{2401}=\log_{7}{7^4}=4\]
